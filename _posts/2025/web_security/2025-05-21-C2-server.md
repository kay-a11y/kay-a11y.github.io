## Mission - Heartbeat Packets

### ğŸ’¡ What is a Beacon (in C2)?

In C2 (Command & Control), **beaconing** means:

> âœ‰ï¸ A client machine (the *agent*) quietly "calls home" to the controller (your laptop) at regular intervals.

This "call" is usually:

* ğŸ“¡ An HTTP/HTTPS request
* ğŸ”‚ Repeating every few seconds or minutes
* ğŸ•µï¸â€â™€ï¸ Invisible to the user
* ğŸ” Potentially carrying system info, GPS, files, commands, etc.

So a "heartbeat packet" is a tiny signal like:

### ğŸ§ª Simple Python Flask Server (laptop side)

```python

app = Flask(__name__)

@app.route('/heartbeat', methods=['POST'])
```

### ğŸ’¡ Example: Fake `whoami` result from Scriptable

```javascript

  let model = Device.model();

    model: model,

```
